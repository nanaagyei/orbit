# ORBIT — LOGO DESIGN BRIEF

## Chosen Style: **Sophisticated Minimalism**

*(A restrained, intelligent blend of styles 1 & 2)*

### Why this style fits Orbit

* YC-backed tools skew **minimal + confident**, not loud
* Your product is about **judgment, reflection, systems thinking**
* Sophisticated minimalism ages well (no redesign in 3–5 years)
* Works equally well:

  * locally
  * open-source
  * enterprise
  * VC-facing

Think:

> **Linear × Notion × Stripe × GitHub**
> not
> flashy AI startups

---

## 1. Brand Personality (Anchor)

Orbit should feel:

* Thoughtful
* Serious
* Calm
* Intentional
* Technical but human

Orbit should *not* feel:

* Trendy
* Noisy
* Mystical
* “AI-first”
* Over-designed

This logo should signal:

> “This tool was built by someone who thinks deeply.”

---

## 2. Core Logo Concept

### Primary Metaphor: **Orbit / Gravity / Connection**

But **abstracted**, not literal.

Key ideas to visually encode:

* Orbits (relationships over time)
* Nodes (people, papers, events)
* Center of gravity (your focus, judgment)
* Compounding motion (growth without chaos)

### DO NOT:

* Draw planets
* Use stars or space clichés
* Use literal network graphs

### INSTEAD:

* Use **suggestive geometry**
* Let the meaning reveal itself subtly

---

## 3. Logo Structure

### A. Primary Mark (Icon)

This is what matters most.

**Shape ideas (pick ONE):**

1. A **broken circle / partial orbit**
2. A **center dot with an implied arc**
3. A **soft ring with an intentional gap**
4. A **minimal node + arc system**

The gap matters:

* Signals openness
* Signals evolution
* Signals non-closure (learning never ends)

> The best logos *imply* motion rather than show it.

---

### B. Wordmark

**Text:** `Orbit`

* Lowercase preferred: `orbit`
* Clean sans-serif
* Neutral weight (not thin, not bold)

Suggested fonts:

* Inter
* Geist
* SF Pro
* Söhne-style fonts

Kerning should feel **precise**, not playful.

---

### C. Combined Lockups

You should have:

1. Icon only
2. Wordmark only
3. Icon + wordmark (horizontal)
4. Icon + wordmark (stacked)

This matters later for:

* README
* GitHub
* Favicon
* App sidebar
* Landing page

---

## 4. Color System

Since the app uses a **white background**, the logo must work in **both light and dark contexts**.

### Primary Color (choose ONE family)

#### Option A — Deep Slate Blue (Recommended)

* Hex range: `#1F2A44` → `#3B4A6B`
* Signals intelligence, trust, depth
* Common among YC fintech / dev tools

#### Option B — Neutral Graphite

* Hex range: `#1C1C1E` → `#3A3A3C`
* Ultra-clean, timeless
* Very GitHub/Stripe-coded

---

### Accent (Optional, Subtle)

Used sparingly:

* Muted blue
* Muted green
* Never neon

Accent is optional — many strong YC logos don’t use one.

---

## 5. Geometry & Proportions

### Design Rules

* Grid-based
* Symmetrical or intentionally asymmetrical
* No decorative strokes
* Rounded edges preferred over sharp

### Stroke Weight

* Medium-light
* Must remain legible at **16×16px**

If it doesn’t work as a favicon, it’s not ready.

---

## 6. Logo Usage Contexts (Design For All)

Your logo must work:

* In a sidebar (small)
* On a README
* As a favicon
* On a landing page hero
* On a pitch deck
* On GitHub

**Test cases to require:**

* Black on white
* White on dark
* Icon-only at 24px
* Icon-only at 512px

---

## 7. Canva / Figma Execution Instructions

If using **Canva**:

* Start from blank canvas
* Use basic shapes only (circle, arc, line)
* No effects, no shadows, no gradients
* Build on a grid
* Export SVG + PNG

If using **Figma**:

* Use vector paths
* Define spacing rules
* Export SVG for dev use

---

## 10. Final Logo Acceptance Checklist

Before you approve a logo, confirm:

* [ ] Works at 16px
* [ ] Works in monochrome
* [ ] Still looks good in 5 years
* [ ] Doesn’t scream “AI startup”
* [ ] Feels calm next to dense UI
* [ ] You’d be proud to show it to a YC partner

If it passes all six — you’re done.

If I were advising you as a founder:
